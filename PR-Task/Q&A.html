Day 1

1.What do you mean by Git and GitHub?
Git is a Version Control System(VCS) where we can keep and track our files and our changes handy. Github is a server which is a platform whiich hosts our files and gives us a ease of public collaboration as well as private collaborations.
2.Why GitHub is so popular and used in most of the projects?
Github seems to be more popular due to its convenient aspects- free, open-source, ease of management.
3.What is a version control system? How Git is a VCS?
VCS is a system which allows tracking,management,sharing,collaborating and storing of multiple versions of a project(in Git a repository). Git is a VCS because it tracks and manages history of different versions of a file.
4.What are the other platforms similar to GitHub?
Other popular platforms similar to Github are GitLab, Bitbucket and SourceForge.
5.Why are you interested in learning of Git and Github?
I am interested in Gi and Github due to its secure storage,seamless version control and immense community.
